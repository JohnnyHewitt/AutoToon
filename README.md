# AutoToon

AI-assisted 2D cartoon production pipeline. Generate assets, storyboards, and automate animation workflows using open-source tools — without generating every frame via AI.

## Getting Started

This project is in **Phase I: Research**. No runnable code yet.

## Phases

1. **Research** — Evaluate open-source 2D animation tools and AI integration techniques
2. **MVP** — Build a simple AI-assisted cartoon creation process
3. **Product** — Package as a web app/tool for public use

## Target Styles

- Modern 2D cartoons (Futurama, Bob's Burgers)
- B/W noir
- General 2D animation
