## Blender Grease Pencil

- 12-min noir short, 100% Grease Pencil pipeline

**Il Baracchino** — 2025, Megadrago (Italy)
- Italy's first adult animated series, Amazon Prime Video
- Closest existing GP production to a Bob's Burgers-style format
- Watch: https://www.primevideo.com/detail/Il-Baracchino/0MA6S7AX667VUDNIHJ78B48LF1
- Production breakdown: https://www.blender.org/user-stories/creating-il-baracchino-italys-first-adult-animated-series-made-with-blender/

## pytvpaint Note

pytvpaint (by BRUNCH Studio) is a Python wrapper for TVPaint via WebSocket RPC.
- **Critical limitation**: Incompatible with TVPaint 12 (current version)
- Only works with TVPaint 11.5
- This may rule out TVPaint for an automated pipeline unless the SDK issue is resolved
- Repo: https://github.com/brunchstudio/pytvpaint
